1. Abrir el proyecto con IntelliJ IDEA 
2. Esperar a que cargue todas las dependencias (Se visualiza en la parte inferior del programa como una barra de carga)
3. Dar clic a boton Run para poder correr la automatización
4. Poder evidenciar la correcta ejecución de este
5. Para visualizar el reporte abrir la carpeta target/karate-reports y abrir src.test.resources.crear-mascota.html (Si no se visualiza correctamente copiar la carpeta karate-reports en el escritorio y desde ahí abrir src.test.resources.crear-mascota.html, adicional puede borrar la carpeta karate-reports y ejecutar el proyecto para que vuelva a generar el reporte y poder abrirlo desde el enlace que sale desde la consola)
